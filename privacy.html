Privacy Policy

AdobeStock Waala MetaData Chrome Extension

Last Updated: January 7, 2025

1. Introduction

Extension Waala ("we," "our," or "us") operates the AdobeStock Waala MetaData Chrome extension. This Privacy Policy explains how we collect, use, and protect your information when you use our extension.

Our Commitment: We are committed to protecting your privacy and ensuring the security of your personal information. This extension is designed with privacy-first principles.

2. Information We Collect

2.1 Authentication Information

  • Login Credentials: Username and password for extension access
  • API Keys: Google Gemini API keys for image analysis functionality
  • Session Data: Login timestamps and session management data

2.2 Usage Preferences

  • Image processing range settings (start/end numbers)
  • Automation preferences (checkbox settings)
  • Subscription status and plan information

2.3 Technical Data

  • Extension version and browser information (for compatibility)
  • Error logs (for debugging and improvement)

3. How We Use Your Information

3.1 Core Functionality

  • Authentication: Verify user identity and manage access
  • Image Analysis: Process images using AI to generate metadata
  • Subscription Management: Track and validate subscription status
  • Settings Persistence: Save and restore user preferences

3.2 Service Improvement

  • Analyze usage patterns to improve functionality
  • Debug and fix technical issues
  • Develop new features based on user needs

4. Data Storage and Security

4.1 Local Storage

  • User preferences and settings stored locally in browser
  • Session data encrypted and stored securely
  • API keys stored locally with encryption

4.2 External Storage

  • Google Sheets: User credentials and subscription data
  • Security Measures: Data encrypted in transit and at rest
  • Access Control: Restricted access to authorized systems only

4.3 Data Retention

  • Local data: Until user logs out or uninstalls extension
  • Server data: Until user requests account deletion
  • Session data: 30 days maximum

5. Third-Party Services

5.1 Google Services

  • Google Gemini AI: Image analysis and metadata generation
  • Google Sheets API: User data management and authentication
  • Google Apps Script: Backend processing and API integration

5.2 Adobe Stock

  • Extension interacts with Adobe Stock contributor pages
  • No data shared with Adobe beyond normal website interaction
  • Image data processed locally before sending to AI service
Important: We do not sell, rent, or share your personal information with third parties for marketing purposes.

6.1 Data Access and Control

  • Access: View your stored data through the extension interface
  • Modification: Update your preferences and settings anytime
  • Deletion: Remove your account and all associated data
  • Export: Request a copy of your data

6.2 Privacy Controls

  • Logout functionality to clear local session data
  • Option to disable automatic data saving
  • Control over which data is shared with AI services

7. Data Processing Legal Basis
